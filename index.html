設定 GitHub Personal Access Token
為什麼需要 Token？
如果您要監控私有 Repository 或避免 GitHub API 速率限制，需要設定 Personal Access Token。
  1. 點擊下方按鈕前往 GitHub
  2. 選擇 "Generate new token (classic)"
  3. 勾選 repo 權限（用於存取私有 repo）
  4. 點擊頁面底部的 "Generate token"
  5. 複製產生的 token（格式：ghp_xxxxx）
前往 GitHub 建立 Token
Token 僅儲存在您的瀏覽器中，不會傳送到任何伺服器
